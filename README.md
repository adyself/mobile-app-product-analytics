# Product Analytics for Mobile E-commerce App

Этот проект моделирует работу продуктового аналитика в мобильном интернет-магазине.

Цель проекта — построить полный аналитический pipeline:

data generation → ETL → аналитические витрины → продуктовая аналитика → BI-дашборд.

## Стек

Python
Pandas
SQL
PostgreSQL
Power BI

## Описание продукта

Анализируемый продукт — мобильное e-commerce приложение для покупки одежды.

Пользователь может:

• открыть приложение
• просматривать товары
• добавлять товары в корзину
• совершать покупку

## Выполненная аналитика

В проекте реализованы:

• расчет продуктовых метрик (DAU, MAU, ARPU, LTV)
• анализ воронки покупок
• когортный анализ retention
• сегментация пользователей
• анализ A/B теста
• интерактивный дашборд Power BI



## Основные инсайты

• выявлены ключевые этапы падения конверсии в воронке
• рассчитан retention пользователей
• оценена эффективность A/B теста
