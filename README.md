# Product Analytics для мобильного e-commerce приложения

Этот проект моделирует работу продуктового аналитика в мобильном интернет-магазине.

Цель проекта — построить полный аналитический пайплайн:
генерация данных → ETL → аналитические витрины → анализ метрик → BI-дашборд.

В проекте используются:
Python, SQL, PostgreSQL, Power BI.

## Описание продукта

Анализируемый продукт — мобильное e-commerce приложение для покупки одежды.

Пользователь может:

- открыть приложение
- просматривать товары
- добавлять товары в корзину
- совершать покупку

## Планируемая аналитика

В проекте реализованы:

- расчет продуктовых метрик (DAU, MAU, ARPU, LTV)
- анализ воронки покупок
- когортный анализ retention
- сегментация пользователей
- анализ A/B теста
- интерактивный дашборд в Power BI
